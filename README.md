# Python Log Analyzer

## About
This project is a simple log analyzer written in Python.
It parses log files and detects suspicious login attempts.

## Features
- Reads log files
- Extracts IP addresses
- Counts failed login attempts
- Detects possible brute-force attacks

## Technologies
- Python
- File handling
- Dictionaries
- Loops and conditions

## Example
Failed login from 192.168.1.10
Failed login from 192.168.1.10

Output:
⚠ Possible brute force detected from 192.168.1.10

## Purpose
Created for learning cybersecurity and SOC fundamentals.
